注意：演示安装的 Mac 版本为 10.13.6

  • github.io 地址：curder/setup-mac-for-developer
  • github 源代码地址：curder/setup-mac-for-developer

开始 ​

bash
git clone https://github.com/curder/setup-mac-for-developer.git
cd setup-mac-for-developer

yarn run docs:dev ## 开发
yarn run docs:build ## 生产

安装 ​

  • Homebrew
  • iTerm2
  • 浏览器
  • Oh-My-Zsh
  • Sublime Text 3
  • PHPStorm
  • nvm|nrm和yarn
  • PHP (可选)
  • Composer
  • Hub
  • Redis (可选)
  • mkcert (可选)
  • Apple M1 上安装 Homestead

配置 ​

  • Homebrew 安装依赖
  • 配置 iTerm2
  • 配置 Oh-My-Zsh
  • 配置 Composer
  • 配置 Git
  • 配置 mkcert
  • 其他
编辑它
